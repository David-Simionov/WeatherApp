🌦️ Weather App

A simple, fast, and user-friendly weather application that provides real-time weather information for any city.

🔗 Live Demo:

https://weather-app-daki.netlify.app/

📌 About the Project

This Weather App allows users to search for any location and instantly get current weather conditions.

The goal of this project is to demonstrate how to work with APIs, handle asynchronous JavaScript, and build a responsive UI for real-world applications.

⚙️ Technologies Used

HTML5 – structure
CSS3 – styling and responsive design
JavaScript (Vanilla JS) – logic and API handling
Weather API – real-time data fetching

✨ Features

🌍 Search by City
Get weather data for any location worldwide
🌡️ Real-Time Weather Data
Temperature
Weather conditions
Other key info (humidity, wind, etc.)
⚡ Fast & Responsive UI
